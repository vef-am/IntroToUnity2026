### About This Project

This project contains hands-on practice with core Unity workflows, including project setup, editor navigation, scene management, GameObject manipulation, materials, lighting, physics, and introductory C# scripting.

### What I Learned

- Set up and configure Unity projects
- Navigate Unity's interface and editor tools
- Create and manipulate GameObjects within scenes
- Transform GameObjects (move, rotate, scale)
- Use parenting to organize hierarchies
- Apply materials to 3D objects
- Create and use Prefabs
- Work with lighting systems
- Implement basic physics
- Write introductory C# scripts

### Tech Stack

- Unity
- C#
- Visual Studio Code

### Repository Structure

- `Assets/`: Project scenes, scripts, materials, and prefabs
- `Packages/`: Unity packages and dependencies
- `ProjectSettings/`: Unity project configuration

### Usage

1. Clone this repository
2. Open the project folder in Unity Hub
3. Launch with the Unity version specified in `ProjectSettings/ProjectVersion.txt`
